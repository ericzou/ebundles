Database Module Not Installed

To use the table browser you must install the appropriate Python database module. See the instructions below for installing the right driver for your DMBS.

MySQL

Download the source from http://sourceforge.net/projects/mysql-pythonAfter you download the file unpack the tar file. You can consult the instructions in the README file. The short version is as follows:

  1. Make sure the command mysql_config is in your PATH. It should be in the same directory as the mysql command.
  2. cd into the directory you unpacked from the tar file
  3. python setup.py build
  4. sudo python setup.py install

When submitting the query to MySQL, the command assumes the following environment variables are properly set:

  • $MYSQL_USER — defaults to the value of $USER
  • $MYSQL_PWD — no default
  • $MYSQL_HOST — defaults to localhost if not set
  • $MYSQL_PORT — defaults to 3306 if not set
  • $MYSQL_DB — no default

These variables may be set in the parent shell of TextMate, global variables in TextMate preferences, or as project local variables.

PostgreSQL

download pgdb from ftp://ftp.pygresql.org/pub/distrib/PyGreSQL.tgz After you have downloaded PyGreSQL and unpacked the tar file, follow the installation instructions Here. The simplified version is as follows:
  1. Make sure pg_config is in your PATH.
  2. sudo python setup.py install

If you are a Postgres user and have the psql8 command installed (as from DarwinPorts for example) the following variables will be used:

  • $PGUSER — defaults to the value of $USER
  • $PGHOST — defaults to localhost if not set
  • $PGPORT — defaults to 3306 if not set
  • $PGDATABASE — defaults to user
